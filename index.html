Avatars

Avatars provides elements which are circular in shape and can be used for profile links, dashboards, and anywhere across the website.

For styling you can use the class avatar and can add avatar-sm, avatar-md and avatar-lg to change avatar's size.

Avatar with letters

Letters Avatars provides text to be inserted inside of Avatar elements instead of images. If an avatar doesn't have a image text can be used as a placeholder for the images.

For styling avatars with text you can use the same classes as the avatar and avatar-bg-s, avatar-bg-p and avatar-bg-n for giving different backgrounds.

Alerts with dismiss

Alerts with dismiss gives a button for users to remove the element from the DOM. You can pass your own onClick handler to the dismiss button.

For proper styling class alert-dismissible can be added to the alert element. For styling class dismiss-btn can be added to dismiss button.

Buttons

Use our custom button styles for actions in forms, dialogs and more you want in your project Buttons have lot of variants like btn-animate for animated buttons.

These are some predefined styles for your buttons each serving its own semantic purpose. You can use them as you want or style them specific to the context it is being used.

Outline Buttons

If you don't want colors on background then you can use btn-outline-* to have buttons with outlines only.

Disabled button

Buttons can be disabled by passing disabled attribute to the button tag.

Link Button

Link button can be used to create buttons which redirect users to a different page or a different section on the same page.

Button with Icons

You can also add icons to the buttons to give more context to the user example for shopping cart by adding btn-icon class to the styles.

It can be used with both normal btn-*(* = primary, secondary, etc) and btn-outline-* as you like

Icon Button

You can have buttons only with icons and all the previous styles can be applied here to (eg. btn-outline-*, btn-*(primary)).

Animated Button

You can use animated buttons for cool effect on hover to your button.

Note: btn-circle should be in the very last to override other btn styles

Images

Images are a very crucial part of any UI. So you can use our image classes to give different styles to your images. Images have have variants like responsive-image to have a responsive-image which changes its size according to its viewport.

Responsive Image

The below example will have an image with an aspect ratio of 16:9 and will respond to changes to viewport width.

Round Images

Round Images will provide images inside an circle container to use across the ui. It should be used with img-container img-circle. It has two variants img-circle and img-circle-sm

Square Images

Square Images will provide images inside an square container to use across the ui. It should be used with img-container img-square.
